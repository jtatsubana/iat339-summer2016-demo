Refined Branding Components

Fonts

A basic sans-serif font with above average amount of curviness. The primary reason for using Source Sans Pro is how approachable its numerical types are. This will come in useful when displaying large amount of statistics. Source Sans Pro will be used as the body font.

Examples : League Table mockup and Team widget mockup


Lato

For headings and larger font sizes, the font Lato will be used.

Colour Palette and Usage

  • Black : #232323 - Texts, Header, and Other Core Elements
  • Green : #135f48 - Graphical Elements, Backgrounds
  • Blue : #0066c0 - Links, etc
  • Gray : #efefef - Background, Subdued Texts

Examples : Match History Table mockup and Stats tables



Imagery

FootyStats will use a subdued color palette. Images will often be overlayed with dark veils as to not "pop" too much.

This is the direction I wanted to go with because FootyStats is intended to be used as a companion to sports betting websites, which often have bright colours as part of their core color scheme.
